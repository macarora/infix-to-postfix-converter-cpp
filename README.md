# Infix to Postfix Converter 🔢

A C++ program that converts infix mathematical expressions 
to postfix notation using a custom stack and queue.

## Features
- Converts infix expressions to postfix notation
- Built using custom linked list stack and queue
- Interactive command-line interface
- Exception handling for invalid expressions

## How to Run
1. Make sure g++ is installed
2. Clone the repo
3. Compile:
```
g++ -o Converter main.cpp infix_to_postfix_converter.cpp
```
4. Run:
```
./Converter
```
5. Type `quit` to exit

## Project Structure
- `main.cpp` — Main driver
- `infix_to_postfix_converter.cpp / .h` — Converter logic
- `lstack.h / .hpp` — Custom linked list stack
- `lqueue.h / .hpp` — Custom linked list queue
